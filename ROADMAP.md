# Roadmap

`Hermes ClawX` 的目标是成为一个稳定、可分发、对普通用户友好的 `Hermes Agent` 桌面客户端。

## Phase 1: Usable Open Source Base

- 明确产品定位和开源协作入口
- 稳定 Electron Main 与 Hermes runtime 托管
- 保持 `lint`、`test`、`build` 持续通过
- 补齐 Issue / PR / Security / License 基础设施

## Phase 2: Core Hermes Workflows

- 强化 Chat 体验
- 强化 Cron 创建、编辑、运行和输出预览
- 强化 Skills 浏览、启停和说明展示
- 强化 Channels 配置保存和错误提示
- 强化 Settings 的导入导出和校验反馈

## Phase 3: Product Polish

- 更完整的首次启动引导
- 更清晰的运行状态、错误态和恢复路径
- 更一致的命名、品牌和 UI 文案
- 更完整的中英文文档

## Phase 4: Distribution and Trust

- 更成熟的三平台打包体验
- 自动更新链路完善
- 发布说明、变更摘要和版本节奏规范化
- 对公开开源用户更友好的文档和示例

## Non-goals

- 不把项目做成泛用的多代理桌面壳
- 不试图伪装成 `ClawX` 官方分支
- 不优先追求大而全，而是优先做好 `Hermes Agent` 的核心工作流
